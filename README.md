# MindCare — Student Mental Health Platform

A comprehensive digital mental health platform built specifically 
for college students. Get AI-powered support, book counseling 
sessions, and connect with a peer community — all in a safe, 
confidential environment.

## Features

- **AI Mental Health Chat** — 24/7 confidential AI support with 
  coping strategies
- **Book Appointments** — Schedule sessions with on-campus 
  counselors
- **Resource Hub** — Videos, audio guides, and articles filtered 
  by category and language
- **Peer Support Community** — Moderated forum with anonymous 
  posting across categories like Anxiety, Depression, Academic 
  Stress
- **Role-based Login** — Separate student and counselor dashboards
- **Crisis Support** — One-click access to Crisis Line: 988

## Getting Started
```bash
# Clone the repository
git clone https://github.com/Anshumaan2006/MindWellness.git

# Navigate to project folder
cd MindWellness

# Install dependencies
npm install

# Start development server
npm run dev
```

Open `http://localhost:5173` in your browser.
